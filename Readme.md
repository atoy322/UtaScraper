# Uta-Net site scraper

## Build
```bash
$ go build
# This operation will create executable file named get_kashi to the current directory.
```

## Usage
```bash
$ ./get_kashi [Song title]
# list of matched songs
[0] xxx
[1] xxx
   :
   :
# input song id
Song id: 0

# save to "{title} - {musician}.txt"
```
